# WorkMate_Task
## Установка
Перед запуском скриптов создайте виртуальное окружение и установите необходимые зависимости через
```bash
pip install -r requirments.txt
```
## Запуск скрипта
```bash
python3 Main.py --files filename.csv --report 
```
## Запуск тестов
```bash
pytest test_main.py
```
## Добавление новых вариантов отчета
Для добавления новых типов отчетов требуется создать класс отчета наследуемый от класса ReportBase и написать метод генерации отчета CreateReport
